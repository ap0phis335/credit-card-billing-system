# credit-card-billing-system
A professional looking credit card billing webapp using flask for backend and html , css , js for frontend

A modern fintech-style web application built using Flask, HTML, CSS, and JavaScript.
This project simulates a basic credit card billing dashboard where users can sign up, log in, add card details, track transactions, and monitor billing information.

Features
 User Signup & Login
 Session Authentication using Flask
 Add Credit Card Details
 Fintech-style Dashboard UI
 Transaction Tracking
 Dynamic Billing Calculations
 Responsive Dark Theme Design
 Smooth Animations & Interactive UI
Tech Stack
 Frontend
   HTML5
   CSS3
   JavaScript
 Backend
   Flask (Python)
